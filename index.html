infra-puppet by jenkinsci

Puppet manifests for managing the Jenkins project infrastructure

Overview

The "infra-puppet" project is an effort to maintain an open source infrastructure for the Jenkins project. This means that everything possible should be entered into Puppet manifests such that the Jenkins developer and infrastructure teams can have a clear idea of what changes are being made/applied, where they're being applied, and why.

Getting Starded

We use Vagrant in order to test changes locally. Installing Vagrant is very simple: 

$ sudo gem install vagrant
Then you can use it to bring up a virtual machine with: 
~/infra-puppet $ vagrant up
While working on changes, you can use the following command to re-apply recently made changes, instead of destroying the virtual machine and starting from scratch: 
~/infra-puppet $ vagrant provision
